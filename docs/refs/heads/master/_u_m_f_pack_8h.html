Usage Example

Problem problem(u, v);
problem = Integral(Grad(u), Grad(v)) - Integral(f, v);

Solver::UMFPack solver(problem);
solver.solve();

Note

This solver requires UMFPACK from SuiteSparse to be installed and RODIN_USE_UMFPACK to be defined at compile time.
