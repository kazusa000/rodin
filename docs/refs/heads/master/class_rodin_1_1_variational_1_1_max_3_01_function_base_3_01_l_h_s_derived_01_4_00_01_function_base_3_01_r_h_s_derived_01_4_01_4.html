Represents the maximum operation between two functions.

Template parameters
LHSDerived Type of the left operand function
RHSDerived Type of the right operand function

Contents

  • Reference
    • Base classes

This class represents the pointwise maximum of two functions:

For scalar functions, this computes the maximum value at each point. For vector or matrix functions, the operation is applied componentwise.

Common applications include:

  • Positive part functions:
  • Barrier methods and penalty functions
  • Non-smooth optimization problems
  • Contact and friction problems in mechanics
