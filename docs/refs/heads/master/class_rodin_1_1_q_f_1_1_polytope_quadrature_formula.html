Rodin::QF::PolytopeQuadratureFormula class
#include <Rodin/QF/PolytopeQuadratureFormula.h>

Generic quadrature formula dispatcher for all polytope types.

Contents

  • Automatic Rule Selection
  • Usage
  • Reference

    • Public functions

PolytopeQuadratureFormula provides a convenient interface for creating quadrature formulas without needing to know which specific quadrature rule to use for each geometry type. It automatically selects an appropriate quadrature formula based on:

  • The polytope geometry (triangle, tetrahedron, etc.)
  • The desired polynomial degree of exactness

Automatic Rule Selection

The class internally selects the most appropriate quadrature rule:

  • For simplices (triangles, tetrahedra): Grundmann-Möller rules
  • For tensor-product elements (quads, hexahedra, wedges): Gauss-Legendre rules
  • For segments: Gauss-Legendre rules
  • For points: Single point with unit weight

Usage

// Create a quadrature rule exact for degree 3 polynomials on a triangle
PolytopeQuadratureFormula qf(3, Geometry::Polytope::Type::Triangle);

// Use default order (degree 1)
PolytopeQuadratureFormula qf_default(Geometry::Polytope::Type::Tetrahedron);
