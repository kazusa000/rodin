Rodin::Geometry::PolytopeIterator class
#include <Rodin/Geometry/PolytopeIterator.h>

Iterator over general polytopes of specified dimension.

Contents

  • Thread Safety
  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

Provides iteration over polytopes of any dimension (0 for vertices, 1 for edges, etc.) in a mesh.

Thread Safety

Not thread-safe. Each thread should use its own iterator instance.

See also

CellIterator, FaceIterator, VertexIterator
