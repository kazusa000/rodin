Rodin/PETSc/Variational/LinearForm.h file

PETSc specialization of variational linear forms.

Contents

  • Mathematical Background
  • Reference

    • Classes

Provides the partial specialization of Rodin::Variational::LinearForm that assembles linear-form contributions into a PETSc Vec. The resulting vector represents the right-hand side of a discrete finite element system .

Mathematical Background

A linear form is a mapping that takes a test function and produces a scalar. After discretisation the action of on the basis functions yields a load vector:

This specialization stores in a PETSc Vec and evaluates via VecDot.
