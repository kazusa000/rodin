Rodin::Solver::LDLT<Math::LinearSystem<Math::Matrix<Scalar>, Math::Vector<Scalar>>> class final

Robust LDLT Cholesky factorization with pivoting for dense matrices.

Template parameters
Scalar The scalar type (e.g., Real, Complex)

Contents

This solver performs LDLT decomposition with robust pivoting for numerical stability. It handles symmetric indefinite matrices and provides reliable solutions for dense systems.

Characteristics:

  • Type: Direct solver
  • Matrix requirements: Symmetric dense (positive definite or indefinite)
  • Memory: for dense storage
  • Stability: Excellent, uses pivoting
