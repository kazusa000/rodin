Rodin::Solid::HyperElasticLaw class

CRTP base class for hyperelastic constitutive laws.

Template parameters
Derived The derived constitutive law type (CRTP)

Contents

  • Reference
    • Public functions

A hyperelastic material is characterized by a stored energy density function from which all stress measures derive.

Each derived class implements:

  • Cache setup: Precomputes law-specific data from the constitutive point.
  • First Piola-Kirchhoff stress:
  • Material tangent: Directional derivative
