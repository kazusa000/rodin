Rodin::IO::HDF5::Path namespace
#include <Rodin/IO/HDF5.h>

HDF5 dataset path constants for the mesh and grid function storage layouts.

Contents

  • Reference
    • Variables

All paths are absolute within their respective HDF5 file. Mesh datasets live under /Mesh/... and grid function datasets under /GridFunction/.... The XDMF-specific derived topology lives under /Mesh/XDMF/....
