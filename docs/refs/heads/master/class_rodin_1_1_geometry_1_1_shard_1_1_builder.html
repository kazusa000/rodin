Rodin::Geometry::Shard::Builder class
#include <Rodin/Geometry/Shard.h>

Builder for constructing Shard instances.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Provides a fluent interface for incrementally building shards from parent meshes with precise control over ownership and ghost polytopes.
