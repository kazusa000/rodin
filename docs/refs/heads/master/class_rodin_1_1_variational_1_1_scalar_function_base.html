Rodin::Variational::ScalarFunctionBase class

Base class for scalar-valued functions with templated scalar type.

Template parameters
Scalar The scalar type (e.g., Real, Complex, Boolean)
Derived The derived class following CRTP pattern

ScalarFunctionBase extends FunctionBase to represent functions mapping points in the domain to scalar values:

where is a scalar field (typically Real, Complex, or Boolean).

Note

This serves as an intermediate base between FunctionBase and concrete scalar function types like RealFunction, ComplexFunction, and BooleanFunction.
