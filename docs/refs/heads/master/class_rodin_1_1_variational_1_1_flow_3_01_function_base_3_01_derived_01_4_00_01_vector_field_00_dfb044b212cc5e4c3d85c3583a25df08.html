Flow map operator for functions.

Template parameters
Derived Type of the function being advected
VectorField Type of the velocity field
Step Time step for flow integration
BoundaryPolicy Policy for handling domain boundaries
TangentPolicy Policy for tangent vector computation

Computes the composition of a function with the flow map generated by a vector field. Given a function and velocity field , this computes where is the flow map.
