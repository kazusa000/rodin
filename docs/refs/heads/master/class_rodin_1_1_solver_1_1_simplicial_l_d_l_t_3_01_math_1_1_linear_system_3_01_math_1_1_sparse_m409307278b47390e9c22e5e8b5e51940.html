Simplicial LDLT Cholesky factorization without square root.

Template parameters
Scalar The scalar type (e.g., Real, Complex)

Contents

  • Reference
    • Base classes

    • Constructors, destructors, conversion operators

This solver performs Cholesky decomposition using a simplicial algorithm. It avoids square root operations, providing better numerical stability than LLT for ill-conditioned matrices.

Characteristics:

  • Type: Direct solver
  • Matrix requirements: Symmetric positive definite sparse
  • Memory: for the factorization
  • Stability: Better than LLT, avoids square roots
