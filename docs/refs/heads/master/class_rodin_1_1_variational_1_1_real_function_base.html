Base class for real-valued scalar functions.

RealFunctionBase extends ScalarFunctionBase with Real as the scalar type, representing functions . This is the most common function type in finite element analysis, used for scalar fields like temperature, pressure, or concentration.

Usage Examples

// Constant real function
RealFunction<Real> f(3.14);

// Lambda-based function
RealFunction<std::function<Real(const Geometry::Point&)>> g(
  [](const Geometry::Point& p) { return p.x() * p.y(); }
);
