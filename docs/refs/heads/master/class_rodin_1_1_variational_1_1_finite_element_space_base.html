FiniteElementSpaceBase provides the foundation for defining finite element spaces where is a function space and is its finite-dimensional approximation. The finite element space defines the basis functions, degrees of freedom, and mesh association required for finite element computations.

const IndexArray& Rodin::Variational::FiniteElementSpaceBase::getDOFs(size_t d, Index i) const pure virtual

Returns the global DOF indices attached to a given mesh entity.

Parameters
d Topological dimension of the entity.
i Index of the entity of dimension d.
Returns Const reference to the array of global DOF indices associated with the finite element on entity .

Let denote the mesh entity of (topological) dimension and index .

This function returns a read-only array of global degree-of-freedom indices corresponding to the local basis functions of the finite element associated with :

  • The entries are ordered exactly as the local basis functions of the finite element returned by getFiniteElement(d, i).
  • The size of the returned array equals the number of local basis functions, i.e. getDOFs(d,i).size() == getFiniteElement(d,i).getCount()
  • For higher-order spaces, the array may contain DOFs that are topologically associated with the entity itself (e.g. edge- or face-interior DOFs) and/or with its subentities (e.g. vertex DOFs), according to the convention of the concrete finite element space.
