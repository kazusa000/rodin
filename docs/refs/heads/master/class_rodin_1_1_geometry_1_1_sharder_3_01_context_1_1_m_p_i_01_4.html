#include <Rodin/MPI/Geometry/Sharder.h>
template<>
Rodin::Geometry::Sharder<Context::MPI> class

Utility for distributing a global mesh across MPI ranks by splitting into per-rank shards, scattering them from a root, and gathering the local mesh on each rank.

One-step distribution: shard, scatter, and gather.

Parameters

template<>
Sharder& Rodin::Geometry::Sharder<Context::MPI>::scatter(int root)

Scatter shards from the root rank to all ranks.

Parameters
root The rank that owns the global mesh and performs sends.
Returns Reference to this object for chaining.

Only the root rank performs non-blocking sends of each shard to its respective owner rank. Other ranks do nothing.

template<>
Mesh<Context::MPI> Rodin::Geometry::Sharder<Context::MPI>::gather(int root)

Gather the local shard on each rank.

Parameters
root The rank that originally scattered the shards.
Returns The local MPI mesh built from the received shard.

On the root rank, returns its own shard without MPI. On other ranks, blocks in a matching MPI_Recv from the root.
