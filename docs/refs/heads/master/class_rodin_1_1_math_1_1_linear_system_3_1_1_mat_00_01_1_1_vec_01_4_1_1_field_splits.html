#include <Rodin/PETSc/Math/LinearSystem.h>
template<>
Rodin::Math::LinearSystem<::Mat, ::Vec>::FieldSplits class

Container for named PETSc index sets used with field-split preconditioners (e.g. PCFIELDSPLIT).

Contents

  • Reference
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

Each Split pairs a human-readable field name (e.g. "velocity", "pressure") with a PETSc IS that identifies the DOF indices belonging to that field within the global system. The class takes ownership of the IS handles and destroys them in its destructor.
