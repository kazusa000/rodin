#include <Rodin/IO/HDF5.h>
template<class FES, class Scalar>
Rodin::IO::GridFunctionLoader<FileFormat::HDF5, FES, Math::Vector<Scalar>> class

HDF5 grid function loader for local grid functions backed by Math::Vector<Scalar>.

Template parameters
FES Finite element space type.
Scalar Scalar element type of the data vector.

Contents

  • Usage Example
  • Reference

Reads a standalone grid function field file (one .h5 per grid function) and populates the DOF vector. The file is expected to contain /GridFunction/Meta/{Size,Dimension} and /GridFunction/Values/Data.

The grid function must already be attached to a finite element space of matching size and dimension before loading.

Stream-based loading is not supported; only file-path loading is available.

Usage Example

P1 Vh(mesh);
GridFunction u(Vh);
IO::GridFunctionLoader<IO::FileFormat::HDF5, P1, Math::Vector<Real>> loader(u);
loader.load("field.h5");
