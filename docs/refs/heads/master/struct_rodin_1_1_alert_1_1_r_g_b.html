Forward declaration of RGB color template.

Template struct for representing RGB colors with compile-time validation. The color components are constrained to the range [0, 255] using static assertions. This enables type-safe custom terminal colors.

Example usage: using MyColor = RGB<128, 64, 255>;

Template struct representing RGB color values for custom terminal colors.
