#include <Rodin/Variational/Average.h>

Rodin::Variational::Average<ShapeFunctionBase<NestedDerived, FES, Space>> class final

Average of a ShapeFunctionBase instance across an interface.

Contents

  • Usage Example
  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

Computes the average of shape (trial or test) functions across element interfaces in finite element formulations. This is used in DG bilinear and linear form assembly.

Usage Example

// DG consistency term
auto consistency = InterfaceIntegral(Average(Grad(u)), Jump(v));
