Base class for scalar-valued functions defined on a mesh.

Represents functions where denotes the field of complex numbers. Complex functions can be:

  • Constructed from real and imaginary parts
  • Decomposed using Re() and Im() operations
  • Conjugated using Conjugate() operation

See also

ComplexFunction, Re, Im, Conjugate
