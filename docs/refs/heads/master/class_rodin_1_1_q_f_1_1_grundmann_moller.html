Implements the Grundmann-Möller integration rules as described in [4]. These quadrature formulae are specifically designed for simplicial elements (segments, triangles, tetrahedra) and provide exact integration for polynomials of odd degree where is a parameter.

Supported Geometries

  • Segment (1D simplex)
  • Triangle (2D simplex)
  • Tetrahedron (3D simplex)

Note

This class only supports simplicial geometries. For tensor-product elements (quadrilaterals, hexahedra), use GaussLegendre or other appropriate rules.

The parameter is limited by RODIN_QF_GRUNDMANNMOLLER_MAX_S.
