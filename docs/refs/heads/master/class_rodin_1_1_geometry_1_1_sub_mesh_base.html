Rodin::Geometry::SubMeshBase class
#include <Rodin/Geometry/SubMesh.h>

Abstract base class for SubMesh implementations.

Contents

  • Reference

Defines the interface for submeshes, which represent subregions of parent meshes while maintaining bidirectional index mappings.
