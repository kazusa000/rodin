Rodin::Alert namespace

Module to handle alerts, exceptions, and diagnostic messages.

Contents

  • Reference
    • Namespaces
    • Classes
    • Enums
    • Typedefs
    • Functions
    • Variables

The Alert module provides a comprehensive system for handling exceptions, warnings, informational messages, and success notifications with colored terminal output and formatted messaging capabilities.

Example usage:

Alert::Info() << "Assembling the system..." << Alert::Raise;
Alert::Success() << "Solution computed successfully." << Alert::Raise;
