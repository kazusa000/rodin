Rodin::Solver::NewtonSolver::Report struct

Summary of the most recent call to solve().

Variable documentation

size_t Rodin::Solver::NewtonSolver::Report::iterations

Number of completed Newton iterations before termination.

This is the zero-based index of the final processed Newton iteration for converged solves. On termination by maximum iterations, it is set to getMaxIterations().

Real Rodin::Solver::NewtonSolver::Report::final_step_norm

It is zero until a linear correction has been computed.

ConvergenceReason Rodin::Solver::NewtonSolver::Report::reason

Reason for termination.
