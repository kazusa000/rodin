Rodin::Assembly::MPI<Rodin::PETSc::Math::LinearSystem, Rodin::Variational::Problem<Rodin::PETSc::Math::LinearSystem, U, V>> class final

MPI-parallel assembly of a single-variable PETSc problem.

Template parameters
U Trial function type.
V Test function type.

Contents

  • Reference
    • Base classes
    • Public types

Each MPI rank assembles its local contributions to the system matrix and right-hand side vector of a Rodin::Variational::Problem backed by PETSc distributed objects.
