#include <Rodin/PETSc/IO/HDF5.h>
template<class FES>
Rodin::IO::GridFunctionPrinter<FileFormat::HDF5, FES, ::Vec> class final

HDF5 printer for PETSc-backed grid functions.

Template parameters
FES Finite element space type.

Contents

  • Reference
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

Writes the local portion of a distributed PETSc vector to an HDF5 file using the same layout expected by GridFunctionLoader.

Note

Stream-based printing is not supported; use the path-based overload.

See also
