Scalar-valued piecewise constant (P0) Lagrange finite element space.

Represents the finite element space composed of discontinuous, piecewise constant functions:

where denotes constant functions on element .

Properties

  • DOF count: One DOF per cell (total: )
  • DOF location: Element barycenter
  • Continuity: Discontinuous across element boundaries ( -conforming)
  • Polynomial degree: 0 (constant functions)
  • Gradient: (zero within each element)

Use Cases

  • Discontinuous Galerkin (DG) methods
  • Flux-based formulations
  • Element-wise constant material properties
  • Finite volume schemes as FEM
  • Pressure space in mixed methods (when inf-sup stability permits)

Example

Mesh Th;
Th = Th.UniformGrid(Polytope::Type::Triangle, {8, 8});
P0 Vh(Th);  // Scalar P0 space
GridFunction u(Vh);
u = 1.0;  // Constant function
