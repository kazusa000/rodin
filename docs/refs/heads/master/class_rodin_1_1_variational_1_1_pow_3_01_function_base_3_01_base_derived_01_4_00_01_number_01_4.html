Represents the power function.

This class represents the exponentiation operation applied to a function:

The base function must be scalar-valued, and the exponent is a compile-time constant. The result is always a real-valued scalar function.

Note

Common use cases include quadratic functions ( ), cubic functions ( ), and fractional powers for regularization.
