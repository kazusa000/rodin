template<size_t K>
Rodin::Variational::WarpFactor1D class

Computes the 1D warp factor for moving equispaced nodes to GLL positions.

Template parameters
K Polynomial degree.

Contents

  • Reference
    • Public static functions

Given an evaluation point , this class computes the warp displacement that interpolates the difference between GLL nodes and equispaced nodes using Lagrange interpolation on the equispaced nodes.

The warp factor is normalized by to ensure smooth blending near the endpoints.

See also

GLL for the target GLL node positions.
