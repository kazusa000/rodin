Rodin::MMG::Adapt class
#include <Rodin/MMG/Adapt.h>

Performs anisotropic/isotropic remeshing from a nodal size map.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

This class wraps MMG adaptation routines and dispatches automatically to:

  • MMG2D for planar 2D meshes,
  • MMG3D for volumetric 3D meshes,
  • MMGS for surface meshes embedded in 3D.

The scalar size-map field encodes target local edge length and is sampled at mesh vertices.

Parameters inherited from MMG5 (hmin/hmax/hausdorff/gradation/angle) can be configured before calling adapt.
