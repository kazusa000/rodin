Rodin/PETSc/Variational/Problem.h file

PETSc specialization of variational problems.

Contents

  • Reference
    • Namespaces
    • Classes

Provides two partial specializations of Rodin::Variational::Problem that assemble into a Rodin::PETSc::Math::LinearSystem:

  1. Two-field (Problem<LinearSystem, U, V>): A single trial / test function pair producing a scalar linear system .
  2. Multi-field (Problem<LinearSystem, U1, U2, U3, Us...>): An arbitrary number of coupled trial / test functions producing a block-structured linear system suitable for field-split preconditioning.

Both specializations support Context::Local (sequential) and Context::MPI (distributed) mesh contexts.

See also

Rodin::PETSc::Variational::TrialFunction, Rodin::PETSc::Variational::TestFunction, Rodin::PETSc::Math::LinearSystem, Rodin::Solver::KSP
