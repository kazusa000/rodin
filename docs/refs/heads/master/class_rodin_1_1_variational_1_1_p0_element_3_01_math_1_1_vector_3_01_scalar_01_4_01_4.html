Piecewise constant (degree 0) vector Lagrange element.

Contents

  • Example
  • Reference
    • Base classes

Vector-valued P0 element with:

  • DOF count: per element (one for each vector component at barycenter)
  • Basis functions: where (unit vectors)
  • Derivatives: (zero gradient)
  • Structure: Component-wise constant, with each basis having one non-zero component

Used for vector-valued DG approximations (e.g., velocity in compressible flow, element-wise constant vector fields).
