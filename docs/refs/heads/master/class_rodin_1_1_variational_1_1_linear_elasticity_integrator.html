Bilinear form integrator for linear elasticity.

FES Finite element space type
LambdaDerived Type of first Lamé parameter function

where is the symmetric strain tensor.

This generic implementation works for any vector-valued finite element space by querying basis function derivatives through the standard FE element interface. Quadrature order is derived automatically from the polynomial order of the FE basis and the Lamé parameter functions.
