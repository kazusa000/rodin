Flow map operator for functions.

Computes the composition of a function with the flow map generated by a vector field. Given a function and a velocity field , this class computes , where is the flow map associated with .

The implementation performs robust cell-by-cell tracing with explicit face crossing detection, adaptive substepping, and optional boundary handling.

Trace Rodin::Variational::Flow<FunctionBase<Derived>, VectorField, Step, BoundaryPolicy><Derived, VectorField, Step, BoundaryPolicy>::trace(const Geometry::Point& p) const

Traces the characteristic starting at p.

Parameters
p Starting point.
Returns Trace result.

The tracing algorithm:

  • consumes the remaining time by repeated cell-local advances,
  • detects the earliest face crossing when needed,
  • hops across interior faces,
  • delegates boundary handling to the boundary policy,
  • aborts safely if it detects non-progress.

< Linearized face hit time.

< Local face index.

< at the start.

< Initial face function value.

< Global face index.
