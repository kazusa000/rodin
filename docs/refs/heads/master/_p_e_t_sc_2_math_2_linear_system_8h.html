Rodin/PETSc/Math/LinearSystem.h file

PETSc specialization of Rodin linear system containers.

Contents

  • Usage
  • Reference
    • Namespaces
    • Classes

Provides the LinearSystem<::Mat, ::Vec> specialization that stores the system matrix , right-hand side vector , and solution vector as PETSc handles. Also defines the Rodin::Math::LinearSystem<::Mat,::Vec>::FieldSplits inner class for block-preconditioner support.

Usage

This class is normally not instantiated directly; instead it is created internally by Rodin::Variational::Problem specializations. Solvers such as Rodin::Solver::KSP receive a reference to the linear system through the solve(LinearSystem&) interface.

See also

Rodin::PETSc::Math::LinearSystem (convenience alias), Rodin::Solver::KSP, Rodin::PETSc::Variational::Problem
