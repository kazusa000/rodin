MFEM printer for PETSc-backed H1 (order ) grid functions.

Template parameters
K Polynomial order.
Range Scalar or vector range type.
Ctx Mesh context type (Local or MPI).

Contents

  • Reference

Handles the Rodin-to-MFEM node reordering using Vandermonde change-of-basis matrices, for both simplex and non-simplex element geometries.
