#include <Rodin/PETSc/Variational/LinearForm.h>
template<class FES>
Rodin::Variational::LinearForm<FES, ::Vec> class final

Linear form specialization that assembles into a PETSc vector.

Template parameters
FES Finite element space type of the associated test function.

Contents

  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

Owns a PETSc Vec representing the load vector of the finite element system. The vector is created in the constructor and destroyed in the destructor.

See also

Rodin::Variational::LinearFormBase, Rodin::PETSc::Variational::LinearForm
