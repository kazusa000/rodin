template<class Context>
Rodin::IO::MeshLoaderBase class

Base class for loading meshes from files or streams.

Contents

  • Usage Example
  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Protected functions

MeshLoaderBase provides the foundation for loading mesh data in different file formats. It extends the generic Loader class to handle mesh-specific operations and manages the mesh object being populated.

Specialized loaders for specific file formats should derive from this class and implement the load(std::istream&) method to parse their respective formats.

Usage Example

Mesh<Context::Local> mesh;
MeshLoader<FileFormat::MFEM, Context::Local> loader(mesh);
loader.load("mesh.mfem");
