Rodin::Solver::SNES class
#include <Rodin/PETSc/Solver/SNES.h>

PETSc SNES (Scalable Nonlinear Equations Solvers) wrapper.

Contents

  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

Wraps the PETSc SNES context for solving nonlinear systems using Newton-type methods. Inherits Rodin::Solver::NewtonSolverBase<KSP> so that the linear sub-problems arising at each Newton step are solved by the associated Rodin::Solver::KSP solver, and Rodin::PETSc::Object<::SNES> for automatic handle cleanup.

Supports both programmatic configuration (setType, setTolerances) and PETSc command-line overrides (-snes_type, -snes_rtol, …).

See also

Rodin::Solver::KSP, Rodin::Solver::NewtonSolverBase, Rodin::PETSc::Math::LinearSystem
