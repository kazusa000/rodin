Rodin::Geometry::BoundedIndexGenerator class final
#include <Rodin/Geometry/IndexGenerator.h>

Index generator for a bounded range of indices.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

Generates indices in the range [start, end), incrementing by 1 each step. This is useful for iterating over consecutive polytope indices.
