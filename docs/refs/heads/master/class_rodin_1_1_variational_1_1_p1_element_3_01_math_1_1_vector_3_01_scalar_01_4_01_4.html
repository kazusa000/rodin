Vector-valued P1 element with:

  • DOF count: where is vector dimension, is vertex count
  • Basis functions:
  • Jacobian: constant per element
  • Continuity: C⁰ continuous vector field

Used for elasticity, fluid mechanics, and vector-valued PDEs. Each component uses P1 interpolation independently.
