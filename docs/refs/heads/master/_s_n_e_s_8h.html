Rodin/PETSc/Solver/SNES.h file

PETSc SNES (Scalable Nonlinear Equations Solvers) wrapper for Rodin.

Contents

  • Reference
    • Namespaces
    • Classes

Wraps the PETSc SNES context for solving nonlinear systems using Newton-type methods. At each Newton iteration the Jacobian system is solved by the associated Rodin::Solver::KSP linear solver.

See also

Rodin::Solver::KSP, Rodin::Solver::NewtonSolverBase, Rodin::PETSc::Math::LinearSystem
