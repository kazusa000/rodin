Encapsulates trial and test finite element spaces along with local and global bilinear form integrators required for assembly.

This class encapsulates all data required to assemble a bilinear form into a matrix operator. It stores references to the trial and test finite element spaces, as well as lists of local and global bilinear form integrators.

Local vs Global Integrators

  • Local integrators: Operate on individual elements , computing contributions for basis functions with support on .
  • Global integrators: Couple multiple elements, useful for non-local operators or interface conditions.
