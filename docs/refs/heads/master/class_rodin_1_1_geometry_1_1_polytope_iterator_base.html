This CRTP (Curiously Recurring Template Pattern) base class provides common iterator functionality for traversing polytopes in a mesh.

Usage Example

// Iterate over all cells in a mesh
for (auto it = mesh.getCell(); it; ++it)
{
  const Cell& cell = *it;
  // Process cell...
}

Thread Safety

Polytope iterators are not thread-safe. Each thread should use its own iterator instance.

See also

PolytopeIterator, CellIterator, FaceIterator, VertexIterator
