Rodin/Variational/BilinearForm.h file

Bilinear form classes for finite element assembly.

Contents

  • Reference
    • Namespaces
    • Classes

This file defines the BilinearForm classes which represent bilinear forms in variational formulations. Bilinear forms are assembled into matrices and form the left-hand side of finite element systems.
