template<class Context>
Rodin::IO::MeshPrinterBase class

Base class for printing meshes to streams or files.

Contents

  • Usage Example
  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

MeshPrinterBase provides the foundation for writing mesh data in different file formats. It extends the generic Printer class to handle mesh-specific operations and manages the mesh object being output.

Specialized printers for specific file formats should derive from this class and implement the print(std::ostream&) method to generate their respective formats.

Usage Example

const Mesh<Context::Local>& mesh = getMesh();
MeshPrinter<FileFormat::MEDIT, Context::Local> printer(mesh);
std::ofstream file("output.mesh");
printer.print(file);
