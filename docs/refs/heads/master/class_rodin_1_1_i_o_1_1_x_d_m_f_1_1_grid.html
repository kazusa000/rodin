Rodin::IO::XDMF::Grid class
#include <Rodin/IO/XDMF.h>

Handle to one named grid inside the XDMF domain.

Contents

  • Reference
    • Public functions

A Grid handle provides access to a specific grid within the XDMF writer. Through this handle, the user can set the mesh, configure export options, and register attributes. Handles are lightweight (pointer + index) and are copyable.
