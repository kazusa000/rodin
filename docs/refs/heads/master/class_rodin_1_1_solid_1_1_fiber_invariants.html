Rodin::Solid::FiberInvariants class
#include <Rodin/Solid/Kinematics/Invariants.h>

Fiber invariants for anisotropic hyperelasticity.

Contents

  • Usage
  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Computes invariants associated with a preferred fiber direction :

Usage

Math::SpatialVector<Real> fiberDir(2);
fiberDir << 1.0, 0.0;
FiberInvariants inv(fiberDir);
inv.setState(state);
Real I4 = inv.getFourthInvariant();
