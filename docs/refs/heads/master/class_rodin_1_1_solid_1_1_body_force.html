Rodin::Solid::BodyForce class
#include <Rodin/Solid/Integrators/BodyForce.h>

Describes a body force per unit reference volume.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

This is a lightweight container for body force data. In a Rodin variational formulation, the user assembles the load vector using:

VectorFunction f{0.0, -9.81};
problem = ...
        - Integral(f, v);

The BodyForce class serves as a semantic wrapper when working within the Solid module's API:

BodyForce body({0.0, -9.81});
// body.getValue() returns the force vector
