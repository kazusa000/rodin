This class implements a boundary handling policy for semi-Lagrangian advection, ensuring that characteristics (particle trajectories) properly interact with domain boundaries.

Mathematical Background

In semi-Lagrangian methods, we trace characteristics backwards in time:

This policy handles cases where characteristics intersect boundaries.
