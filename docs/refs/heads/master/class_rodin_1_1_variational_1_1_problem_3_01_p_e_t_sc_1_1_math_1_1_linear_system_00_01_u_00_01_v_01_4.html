#include <Rodin/PETSc/Variational/Problem.h>
template<class U, class V>
Rodin::Variational::Problem<PETSc::Math::LinearSystem, U, V> class

PETSc variational problem for a single trial / test function pair.

Template parameters
U Trial function type (e.g. TrialFunction<GridFunction<P1<…>>, P1<…>>).
V Test function type (e.g. TestFunction<P1<…>>).

Contents

  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

Assembles a variational formulation into a Rodin::PETSc::Math::LinearSystem containing the system matrix , the right-hand side , and the solution . After calling solve(), the solution is automatically scattered back into the trial function's grid function.

See also

Rodin::PETSc::Variational::Problem (convenience alias), Rodin::Solver::KSP
