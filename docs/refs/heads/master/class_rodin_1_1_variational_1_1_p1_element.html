Continuous piecewise linear (degree 1) scalar Lagrange element.

Note

For an overview of all the possible specializations of the P1 class, please see P1 Template Specializations.

See also

P1Element Template Specializations

The P1Element provides a first-order finite element with:

  • DOF count: One per vertex ( total)
  • Basis functions: Linear functions satisfying
  • Gradient: Constant on each element,
  • Continuity: Global C⁰ continuity across element interfaces

P1 elements provide first-order convergence and are the standard choice for elliptic PDEs like Poisson's equation .
