template<size_t K>
Rodin::Variational::GLL class

Compile-time Gauss–Lobatto–Legendre (GLL) nodes on [-1, 1].

Template parameters
K Polynomial degree.

Contents

  • Reference
    • Public static functions

This class provides the GLL nodes associated with the Legendre polynomial degree . The nodes consist of the endpoints and the interior zeros of .

The nodes are computed once at compile time (subject to compiler support for constexpr floating-point operations) using a Newton iteration with Chebyshev initial guesses, and cached in a static constexpr array.
