Handles assembly of multiple bilinear forms organized as a tuple, typically used for mixed or block formulations.

This class handles assembly of multiple bilinear forms organized as a tuple, typically used for mixed finite element formulations or block systems. It manages the layout and offsets for assembling block matrices.
