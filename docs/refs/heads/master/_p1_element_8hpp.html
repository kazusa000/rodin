Template implementation of P1 basis functions and derivatives.

This file provides the explicit implementations of P1 basis functions and their first derivatives for each supported polytope geometry.

Basis Functions by Geometry

Point (0D):

Segment (1D): Linear interpolation on

Triangle (2D): Barycentric coordinates on reference triangle

Quadrilateral (2D): Bilinear functions on

Tetrahedron (3D): Barycentric coordinates

Wedge (3D): Product of triangle and segment basis

All derivatives are computed analytically as the partial derivatives of these basis functions.
