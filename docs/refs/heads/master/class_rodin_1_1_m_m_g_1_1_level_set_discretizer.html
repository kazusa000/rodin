Rodin::MMG::LevelSetDiscretizer class
#include <Rodin/MMG/LevelSetDiscretizer.h>

Discretizes implicit geometry defined by a nodal level-set field.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

Given a scalar function on a background mesh , this class drives MMG level-set mode to construct either:

  • a boundary-fitted volume mesh for one side of , or
  • an explicit surface mesh of the isovalue when surface is enabled.

It also supports material-splitting policies (see SplitMap) so multi-material meshes can be split into interior/exterior regions with user-defined labels.

The canonical signed-distance interpretation is:

Inherits common remeshing controls from MMG5 (hmin/hmax/hausdorff/ gradation/angle detection).
