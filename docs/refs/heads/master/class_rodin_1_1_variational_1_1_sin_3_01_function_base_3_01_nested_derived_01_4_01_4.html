Rodin::Variational::Sin<FunctionBase<NestedDerived>> class final

Sine function operator for real-valued scalar functions.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

Applies the sine function pointwise to a given function:

Common applications include:

  • Periodic boundary conditions
  • Wave equations and vibration problems
  • Fourier series representations
  • Trigonometric manufactured solutions

Note

Always returns a real-valued function for real input.
