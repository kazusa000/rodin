#include <Rodin/PETSc/Variational/BilinearForm.h>
template<class Solution, class TrialFES, class TestFES>
Rodin::Variational::BilinearForm<Solution, TrialFES, TestFES, ::Mat> class final

Bilinear form specialization that assembles into a PETSc matrix.

Template parameters
Solution Solution grid function type for the trial function.
TrialFES Finite element space type of the trial function.
TestFES Finite element space type of the test function.

Contents

  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

Owns a PETSc Mat representing the system matrix . The matrix is created in the constructor (using an MPI communicator deduced from the trial and test meshes) and destroyed in the destructor.

See also

Rodin::Variational::BilinearFormBase, Rodin::PETSc::Variational::BilinearForm
